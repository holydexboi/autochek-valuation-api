## Autochek Vehicle Valuation API

### Prerequisites

- Node.js
- NPM

### Installation

1. Clone the repository:

   ```bash
   $ git clone https://github.com/your-repo/autochek-valuation-api.git

   $ cd autochek-valuation-api
   ```

### Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
